Programación Web

HTML

Es un lenguaje utilizado para el desarrollo de páginas web.
Podemos utilizar cualquier editor de texto para editar o manipular el contenido de un documento HTML.

JavaScript

Es un lenguaje que no permite crear pequeños programas que se insertan en páginas web.
JavaScript nos permite crear eectos dinámicos para nuestras páginas web.

JQuery

Son librerías y ficheros de JavaScript, que nos facilitan ejecutar tareas en paginas web
JQuery nos permite implementar procesos ya listos para usar, facilitandonos la solucion de tareas básicas
